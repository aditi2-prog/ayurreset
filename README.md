# AyurReset

Personalized Ayurveda Gut & Energy Reset Program.

## Setup Instructions

### 1. Environment Variables
Create a `.env` file in the root directory and add the following variables:

```env
# Razorpay Configuration
RAZORPAY_KEY_ID=your_razorpay_key_id
RAZORPAY_KEY_SECRET=your_razorpay_key_secret
```

### 2. Installation
```bash
npm install
```

### 3. Development
```bash
npm run dev
```

### 4. Build
```bash
npm run build
```

## Deployment
This app is designed to be deployed on platforms like Cloud Run or Vercel. Ensure you set the environment variables in your deployment platform's settings.
